# CLI Productivity Tracker

A simple command-line productivity tracker built using Python.
Manage tasks directly from the terminal with persistent JSON storage.

## Features
- Add, list, complete, and delete tasks
- Persistent storage using JSON
- Clean CLI built with argparse

## Tech Stack
- Python
- argparse
- JSON
